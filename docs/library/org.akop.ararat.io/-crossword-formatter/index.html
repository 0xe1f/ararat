interface CrosswordFormatter

The CrosswordFormatter interface provides a means to read and write crosswords to/from streams.

Sets the stream encoding for subsequent reads/writes. Almost all formatters are expected to have a default encoding, and not all will support all encodings, due to format restrictions or other reasons.
